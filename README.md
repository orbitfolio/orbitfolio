# Orbit Folio Starter

Beautiful, serverless portfolio tracker for stocks, crypto, gold, MFs, real estate, & NFTs. Built with Next.js 15, Tailwind, Chart.js. Dark/light modes, global currencies (USD base), SmartSearch, CSV upload.
